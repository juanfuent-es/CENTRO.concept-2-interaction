Introducción

Esta actividad propone el uso de prompts, intuición y criterio creativo para desarrollar una interacción dentro del browser.

No se trata de construir una web compleja, sino una micro interacción memorable por medio de asistencia de IA.

Objetivo principal

Demostrar el impacto de las interacciones en la creación de un producto digital.

Deadline: Martes 5 de mayo

Conceptualización

Antes de diseñar, debes responder:

  • ¿Qué idea quieres comunicar?
  • ¿Qué emoción o atmósfera quieres generar?
  • ¿Por qué esa estética?
  • ¿Por qué esa interacción?
  • ¿Qué hace única tu propuesta?

Ejemplos válidos

  • Poster/Hero digital reactivo
  • Tipografía/Logotipo interactivo digital
  • Interfaz inspirada en sonido
  • Tecnologías Web Experimentales
  • Estados de botón interactivos

Interacción

La pieza debe responder a la acción del usuario.. No reprooducciones automáticas, loops infinitos o animaciones sin interacción.

  • Hover con respuesta visual
  • Mouse move reactivo
  • Click que transforma elementos
  • Teclado
  • Audio reactivo
  • Scroll narrativo
  • Reveal progresivo
  • Generación aleatoria controlada

Se valorará más el riesgo creativo que la solución mínima.

Sistema Visual

  • Jerarquía tipográfica clara
  • Paleta intencional
  • Consistencia visual
  • Uso de imagen, motion o textura con criterio
  • Relación clara entre forma y concepto
  • Buen ritmo visual en pantalla

Implementación Técnica

  • Código legible
